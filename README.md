## :pushpin:작성 규칙
- 매주 `일요일` 차주에 공부 할 사항들을 markdown에 기록 
- 월, 수, 금 자신의 학습 리스트를 확인하면서 체크 후 `Pull Request` 요청
- 자신의 branch\year\week
  ```text
    ├── README.md
    ├── sjchoi
    │   ├── 2021
    │   │   └── Nov
    │   │       └── 4th-week.md
    │   └── README.md
  ```
- 체크박스 형태로 해당 주에 할 공부내용 기록
    ```text
      ## 11월 4주차 
      - [ ] toast-api 프로젝트 적용하기
      - [x] Mixin 적용
      - [x] Earth (Orbit/Moon)
    ```
    ## 11월 4주차 
    - [ ] Swift 공부
    - [x] Javascript Study
    - [ ] Spring boot - migration
